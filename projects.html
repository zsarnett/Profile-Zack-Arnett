Lawler Discord Bot

The Lawler Discord Bot is a C# .NET-Core Bot that uses the Discord API to send and receive messages. Its intended to be for Lawler to use when he does't want to type out lengthy messages that he uses frequently. The bot also connects to the Twitch API to check when Lawler is streaming and automatically send a message with the stream title, game and link to the stream.
